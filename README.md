## Install and use

- Get the project code

```bash
git clone https://github.com/dgiot/dgiot_dashboard.git
```

- Installation dependencies

```bash
cd dgiot_dashboard

npm install cnpm -g --registry=https://registry.npm.taobao.org

cnpm install

```

- run

```bash
cnpm run serve
```

- build

```bash
cnpm run build
```
